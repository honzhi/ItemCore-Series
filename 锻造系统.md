## 核心类实现

### 1. QueueItem - 队列项（最简单的开始）

这是队列系统的基础，存储每个正在锻造的物品信息。

```java
package com.example.forge.core;

import java.util.UUID;

/**
 * 锻造队列中的单个项
 * 使用时间戳计算状态，无需定时任务
 */
public class QueueItem {
    
    private final String recipeId;       // 配方ID
    private final UUID uuid;             // 唯一标识
    private final long start;            // 开始时间戳
    private long completion;             // 完成时间戳（可变）
    
    public QueueItem(String recipeId, UUID uuid, long start, long completion) {
        this.recipeId = recipeId;
        this.uuid = uuid;
        this.start = start;
        this.completion = completion;
    }
    
    /**
     * 是否已完成
     */
    public boolean isReady() {
        return getRemaining() <= 0;
    }
    
    /**
     * 获取剩余时间（毫秒）
     */
    public long getRemaining() {
        return Math.max(0, completion - System.currentTimeMillis());
    }
    
    /**
     * 获取已用时间（毫秒）
     */
    public long getElapsed() {
        return System.currentTimeMillis() - start;
    }
    
    /**
     * 提前完成时间（当前面的项被取消时）
     */
    public void advanceCompletion(long amount) {
        this.completion -= amount;
    }
    
    // Getters
    public String getRecipeId() { return recipeId; }
    public UUID getUuid() { return uuid; }
    public long getStart() { return start; }
    public long getCompletion() { return completion; }
}
```

### 2. CraftingQueue - 锻造队列

管理单个锻造台的队列，处理添加、移除和时间计算。

```java
package com.example.forge.core;

import java.util.ArrayList;
import java.util.Collections;
import java.util.List;
import java.util.Optional;
import java.util.UUID;

/**
 * 单个锻造台的锻造队列
 */
public class CraftingQueue {
    
    private final String stationId;
    private final List<QueueItem> items = new ArrayList<>();
    
    public CraftingQueue(String stationId) {
        this.stationId = stationId;
    }
    
    /**
     * 检查队列是否已满
     */
    public boolean isFull(int maxSize) {
        return items.size() >= maxSize;
    }
    
    /**
     * 添加新配方到队列
     * 关键逻辑：计算完成时间
     */
    public QueueItem add(String recipeId, double craftingTimeSeconds) {
        long craftingTimeMs = (long) (craftingTimeSeconds * 1000);
        long start = System.currentTimeMillis();
        
        // 如果队列为空：完成时间 = 当前时间 + 配方时间
        // 如果队列不为空：完成时间 = 上一个完成时间 + 当前配方时间
        long completion = items.isEmpty() 
            ? start + craftingTimeMs 
            : items.get(items.size() - 1).getCompletion() + craftingTimeMs;
        
        QueueItem item = new QueueItem(
            recipeId, 
            UUID.randomUUID(), 
            start, 
            completion
        );
        items.add(item);
        return item;
    }
    
    /**
     * 领取（移除已完成的项，不影响后续时间）
     */
    public void removeForClaim(UUID itemId) {
        int index = findIndex(itemId);
        if (index < 0) return;
        items.remove(index);
        // 领取已完成的项，不影响后续时间
    }
    
    /**
     * 取消（移除未完成的项，后续项时间提前）
     * 关键逻辑：后续项时间提前
     */
    public void removeForCancel(UUID itemId, double craftingTimeSeconds) {
        int index = findIndex(itemId);
        if (index < 0) return;
        
        QueueItem removed = items.remove(index);
        
        // 计算节省的时间 = min(剩余时间, 配方时间)
        long craftingTimeMs = (long) (craftingTimeSeconds * 1000);
        long savedTime = Math.min(removed.getRemaining(), craftingTimeMs);
        
        // 后续所有项的完成时间提前
        for (int i = index; i < items.size(); i++) {
            items.get(i).advanceCompletion(savedTime);
        }
    }
    
    /**
     * 获取指定UUID的队列项
     */
    public Optional<QueueItem> getItem(UUID uuid) {
        return items.stream()
            .filter(item -> item.getUuid().equals(uuid))
            .findFirst();
    }
    
    private int findIndex(UUID itemId) {
        for (int i = 0; i < items.size(); i++) {
            if (items.get(i).getUuid().equals(itemId)) {
                return i;
            }
        }
        return -1;
    }
    
    // Getters
    public List<QueueItem> getItems() { return Collections.unmodifiableList(items); }
    public int size() { return items.size(); }
    public String getStationId() { return stationId; }
}
```

### 3. CraftingStatus - 玩家锻造状态

管理玩家所有锻造台的队列。

```java
package com.example.forge.core;

import org.bukkit.configuration.ConfigurationSection;
import java.util.HashMap;
import java.util.Map;
import java.util.UUID;

/**
 * 玩家锻造状态
 * 每个玩家一个实例，管理该玩家所有锻造台的队列
 */
public class CraftingStatus {
    
    // 按锻造台ID分类存储队列
    private final Map<String, CraftingQueue> queues = new HashMap<>();
    
    /**
     * 获取或创建指定锻造台的队列
     */
    public CraftingQueue getQueue(String stationId) {
        return queues.computeIfAbsent(stationId, CraftingQueue::new);
    }
    
    /**
     * 保存到配置
     */
    public void save(ConfigurationSection section) {
        for (Map.Entry<String, CraftingQueue> entry : queues.entrySet()) {
            String stationId = entry.getKey();
            CraftingQueue queue = entry.getValue();
            
            for (QueueItem item : queue.getItems()) {
                String path = stationId + "." + item.getUuid().toString();
                section.set(path + ".recipe", item.getRecipeId());
                section.set(path + ".start", item.getStart());
                section.set(path + ".completion", item.getCompletion());
            }
        }
    }
    
    /**
     * 从配置加载
     */
    public void load(ConfigurationSection section) {
        for (String stationId : section.getKeys(false)) {
            CraftingQueue queue = getQueue(stationId);
            
            ConfigurationSection stationSection = section.getConfigurationSection(stationId);
            for (String itemKey : stationSection.getKeys(false)) {
                String recipeId = stationSection.getString(itemKey + ".recipe");
                long start = stationSection.getLong(itemKey + ".start");
                long completion = stationSection.getLong(itemKey + ".completion");
                
                // 直接添加到队列（不重新计算时间）
                QueueItem item = new QueueItem(
                    recipeId, 
                    UUID.fromString(itemKey), 
                    start, 
                    completion
                );
                queue.getItems().add(item);  // 注意：这里需要在CraftingQueue添加方法
            }
        }
    }
}
```

### 4. TimeFormatter - 时间格式化工具

```java
package com.example.forge.util;

/**
 * 时间格式化工具
 */
public class TimeFormatter {
    
    private static final long[] UNITS = {
        24 * 60 * 60 * 1000,  // 天
        60 * 60 * 1000,       // 时
        60 * 1000,            // 分
        1000                  // 秒
    };
    
    private static final String[] SUFFIXES = {"d", "h", "m", "s"};
    
    /**
     * 格式化剩余时间（最多显示两个单位）
     */
    public static String format(long ms) {
        if (ms <= 0) return "0s";
        
        StringBuilder result = new StringBuilder();
        int count = 0;
        
        for (int i = 0; i < UNITS.length && count < 2; i++) {
            if (ms >= UNITS[i]) {
                long value = ms / UNITS[i];
                result.append(value).append(SUFFIXES[i]).append(" ");
                ms %= UNITS[i];
                count++;
            }
        }
        
        return result.length() > 0 ? result.toString().trim() : "1s";
    }
    
    // 示例:
    // format(3661000) -> "1h 1m"
    // format(150000)  -> "2m 30s"
    // format(5000)    -> "5s"
}
```

## 队列操作流程图

### 添加到队列

```
玩家点击配方
     │
     ▼
┌─────────────────┐
│ 检查材料/条件   │
└────────┬────────┘
         │ 不满足
         ├─────────► 返回错误
         │ 满足
         ▼
┌─────────────────┐
│ 扣除材料        │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ craftingTime=0? │
└────┬──────┬─────┘
     │是    │否
     ▼      ▼
┌────────┐ ┌─────────────┐
│ 即时   │ │ 队列满?     │
│ 完成   │ └──────┬──────┘
└────────┘        │满
                   ├────────► 队列已满
                   │不满
                   ▼
            ┌─────────────┐
            │ 计算完成时间│
            │ 添加到队列  │
            └─────────────┘
```

### 时间计算示例

```
场景1: 空队列添加配方A（耗时10秒）
时间线:
当前:  1000ms
start: 1000ms
completion: 1000 + 10000 = 11000ms

场景2: 在A完成前添加配方B（耗时5秒）
时间线:
当前:  3000ms (A已用2秒，剩余8秒)
B的start: 3000ms
B的completion: A.completion + 5000 = 11000 + 5000 = 16000ms

场景3: 取消配方A（此时A还剩8秒）
时间线:
移除A后，节省的时间 = min(8000, 10000) = 8000ms
B的completion = 16000 - 8000 = 8000ms
```

### 领取和取消

```
领取已完成的物品:
1. 从队列移除
2. 给予物品
3. 执行 on-claim 触发器

取消未完成的物品:
1. 从队列移除
2. 后续项时间提前
3. 返还材料
4. 执行 on-cancel 触发器
```

### VanillaIngredient - 原版材料实现

```java
package com.example.forge.ingredient;

import org.bukkit.Material;
import org.bukkit.entity.Player;
import org.bukkit.inventory.ItemStack;
import org.bukkit.inventory.PlayerInventory;

/**
 * 原版物品材料
 */
public class VanillaIngredient implements Ingredient {
    
    private final Material material;
    private final int amount;
    
    public VanillaIngredient(Material material, int amount) {
        this.material = material;
        this.amount = amount;
    }
    
    @Override
    public boolean hasEnough(Player player) {
        PlayerInventory inv = player.getInventory();
        int count = 0;
        for (ItemStack item : inv.getContents()) {
            if (item != null && item.getType() == material) {
                count += item.getAmount();
            }
        }
        return count >= amount;
    }
    
    @Override
    public void consume(Player player) {
        PlayerInventory inv = player.getInventory();
        int remaining = amount;
        
        for (int i = 0; i < inv.getSize() && remaining > 0; i++) {
            ItemStack item = inv.getItem(i);
            if (item != null && item.getType() == material) {
                if (item.getAmount() <= remaining) {
                    remaining -= item.getAmount();
                    inv.setItem(i, null);
                } else {
                    item.setAmount(item.getAmount() - remaining);
                    remaining = 0;
                }
            }
        }
    }
    
    @Override
    public void returnTo(Player player) {
        player.getInventory().addItem(new ItemStack(material, amount));
    }
    
    @Override
    public ItemStack getDisplayItem() {
        return new ItemStack(material, amount);
    }
    
    @Override
    public String getDescription() {
        return amount + "x " + material.name();
    }
    
    // 从字符串解析: "vanilla IRON_INGOT 5"
    public static VanillaIngredient parse(String format) {
        String[] parts = format.split(" ");
        if (parts.length < 3) {
            throw new IllegalArgumentException("Invalid ingredient format: " + format);
        }
        Material material = Material.matchMaterial(parts[1]);
        if (material == null) {
            throw new IllegalArgumentException("Invalid material: " + parts[1]);
        }
        int amount = Integer.parseInt(parts[2]);
        return new VanillaIngredient(material, amount);
    }
}
```

```java
public class ForgePlugin extends JavaPlugin {
    
    private static ForgePlugin instance;
    
    private StationManager stationManager;
    private RecipeManager recipeManager;
    private PlayerDataManager playerDataManager;
    
    @Override
    public void onEnable() {
        instance = this;
        
        // 保存默认配置
        saveDefaultConfig();
        
        // 初始化管理器
        stationManager = new StationManager(this);
        recipeManager = new RecipeManager(this);
        playerDataManager = new PlayerDataManager(this);
        
        // 加载配置
        stationManager.load();
        recipeManager.load();
        
        getLogger().info("ForgePlugin 已启用！");
    }
    
    @Override
    public void onDisable() {
        // 保存玩家数据
        playerDataManager.saveAll();
        
        getLogger().info("ForgePlugin 已禁用！");
    }
    
    public static ForgePlugin getInstance() {
        return instance;
    }
    
    public StationManager getStationManager() { return stationManager; }
    public RecipeManager getRecipeManager() { return recipeManager; }
    public PlayerDataManager getPlayerDataManager() { return playerDataManager; }
}
```

## 关键注意事项

### 1. 时间戳只存储绝对值

```
正确的做法:
- 存储: start (开始时间戳), completion (完成时间戳)
- 计算: remaining = max(0, completion - System.currentTimeMillis())

错误的做法:
- 存储: remaining (剩余时间)
- 问题: 服务器重启后时间不会继续流逝
```

### 2. 取消时的时间调整

```
当取消队列中的第 N 项时：
1. 计算该项原本需要的时间（recipe.craftingTime）
2. 计算该项实际剩余时间
3. 取较小值作为"节省的时间"
4. 后续所有项的 completion 减去这个值

为什么取较小值？
- 如果项刚开始不久，实际节省的时间 = 剩余时间
- 如果项已经完成了大部分，实际节省的时间 ≈ 配方时间
```

### 3. 即时配方和队列配方的区别

```
即时配方（crafting-time <= 0）:
├── 不添加到队列
├── 直接给予物品
├── 无法取消

队列配方（crafting-time > 0）:
├── 添加到队列
├── 需要等待
├── 可以取消
└── 领取时给予物品
```

### 4. GUI刷新策略

```
不要使用全局定时任务刷新所有玩家的GUI

正确做法:
- 只在玩家打开锻造台GUI时启动刷新任务
- 每20 ticks（1秒）刷新一次队列显示
- 当GUI关闭时自动取消任务

示例:
new BukkitRunnable() {
    public void run() {
        if (inventory.getViewers().isEmpty()) {
            cancel();
            return;
        }
        // 更新队列显示
    }
}.runTaskTimer(plugin, 0, 20);
```

### 5. 数据持久化

```
玩家数据存储格式:
players/<uuid>.yml
└── crafting:
    └── <station-id>:
        └── <queue-item-uuid>:
            ├── recipe: <recipe-id>
            ├── start: <timestamp>
            └── completion: <timestamp>

加载时:
- 直接读取 start 和 completion
- 不做任何时间调整
- 让 isReady() 和 getRemaining() 实时计算
```

## 测试用例

### 测试1: 基本队列操作

```
1. 玩家添加配方A（10秒）
   验证: 队列大小 = 1, A.completion = now + 10000

2. 玩家添加配方B（5秒）
   验证: 队列大小 = 2, B.completion = A.completion + 5000

3. 等待10秒后
   验证: A.isReady() = true, B.getRemaining() ≈ 5000

4. 领取A (使用 removeForClaim)
   验证: 队列大小 = 1, B.completion 不变（A已完成，不影响后续）
```

### 测试2: 取消中间项

```
1. 添加A(10s), B(5s), C(8s)
   队列: [A, B, C]
   完成时间: A=10s, B=15s, C=23s

2. 等待3秒后取消A (使用 removeForCancel)
   节省时间 = min(7s, 10s) = 7s
   
3. 验证:
   B的完成时间 = 15s - 7s = 8s
   C的完成时间 = 23s - 7s = 16s
```

### 测试3: 服务器重启

```
1. 添加配方A(60秒)
   start = T0, completion = T0 + 60000

2. 等待10秒后，触发插件保存 (CraftingStatus.save)
   保存的数据:
   - start: T0 (绝对时间戳)
   - completion: T0 + 60000 (绝对时间戳)

3. 重启服务器，重新加载数据 (CraftingStatus.load)
   注意: 不调整任何时间，直接读取原值

4. 服务器重启后 (假设重启用了5秒)
   当前时间 = T0 + 15000ms
   
   验证:
   - A.getRemaining() = max(0, (T0+60000) - (T0+15000)) = 45000ms
   - 时间计算完全正常，重启不影响
```

**关键要点：只存储绝对时间戳，不存储相对时间**
