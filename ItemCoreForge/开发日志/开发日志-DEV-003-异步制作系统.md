### 1. CraftScheduler 制作调度器

**核心数据结构**：
```java
public class CraftScheduler {
    private final Map<UUID, BukkitTask> activeTasks;  // 活跃任务
    private final Map<UUID, Long> taskEndTimes;       // 任务结束时间
    private BukkitTask updateTask;                    // 更新任务
}
```

**制作流程**：
```
enqueueCraft(player, forgeId, recipeId)
    ↓
1. 验证锻造台和配方存在
    ↓
2. 检查队列是否已满
    ↓
3. 检查材料是否足够
    ↓
4. 检查条件是否满足
    ↓
5. 消耗材料
    ↓
6. 创建 CraftTask
    ↓
7. 加入队列
    ↓
8. 如果是当前任务，启动异步执行
    ↓
9. 发送开始制作消息
```

### 2. 异步任务执行

**启动任务**：
```java
private void startCraftTask(Player player, CraftTask task, RecipeQueue queue) {
    long durationTicks = (long) (task.getDuration() / 50);
    
    BukkitTask bukkitTask = Bukkit.getScheduler().runTaskLater(plugin, () -> {
        executeCraft(player, task, queue);
    }, durationTicks);
    
    activeTasks.put(task.getTaskId(), bukkitTask);
    taskEndTimes.put(task.getTaskId(), System.currentTimeMillis() + task.getDuration());
}
```

### 3. 队列时间管理

**获取剩余时间**：
```java
public long getRemainingTime(UUID taskId) {
    Long endTime = taskEndTimes.get(taskId);
    if (endTime == null) {
        return 0;
    }
    return Math.max(0, endTime - System.currentTimeMillis());
}

public double getRemainingSeconds(UUID taskId) {
    return getRemainingTime(taskId) / 1000.0;
}
```
