# 开发日志 - DEV-021 - 制作完成时立即刷新

**开发日期**: 2026-05-27  
**开发阶段**: Phase 21  
**开发状态**: ✅ 已完成

---

## 📋 开发目标

确保物品制作完成后能够立即刷新队列显示。

---

## 🔧 问题描述

**用户反馈**: 当有物品制作完成后也需要立即刷新一次。

**原因分析**:

虽然在 `executeCraft()` 中已经触发了 `RecipeQueueEvent`，但事件系统可能存在时序问题或配置问题，导致 GUI 没有立即更新。

---

## 🛠 修复方案

采用与 `startCraft()` 相同的 **直接调用** 机制，确保制作完成后立即刷新。

---

## 📝 具体实现

### 修改文件 1: `gui/GuiListener.java`

#### 新增方法 `updatePlayerQueueDisplay()`

```java
public void updatePlayerQueueDisplay(Player player) {
    BaseMenu menu = openMenus.get(player.getUniqueId());
    if (menu instanceof QueueUpdateable) {
        ((QueueUpdateable) menu).updateQueueDisplay();
    }
}
```

**作用**: 提供一个公共方法，允许其他模块直接通知特定玩家的 GUI 更新队列显示。

### 修改文件 2: `task/CraftScheduler.java`

#### 在 `executeCraft()` 中添加直接调用

```java
// 修复前
RecipeQueueEvent queueEvent = new RecipeQueueEvent(player, forge, queue);
Bukkit.getPluginManager().callEvent(queueEvent);

CraftTask nextTask = queue.getCurrentTask();
if (nextTask != null) {
    startCraftTask(player, nextTask, queue);
}

// 修复后
RecipeQueueEvent queueEvent = new RecipeQueueEvent(player, forge, queue);
Bukkit.getPluginManager().callEvent(queueEvent);

// 直接通知 GUI 更新
plugin.getGuiListener().updatePlayerQueueDisplay(player);

CraftTask nextTask = queue.getCurrentTask();
if (nextTask != null) {
    startCraftTask(player, nextTask, queue);
}
```

| 操作 | 修复前 | 修复后 |
|------|--------|--------|

## 📊 完整更新机制总结

现在队列会在以下时机更新：

### 1. 开始制作时

```java
// CustomForgeGUI.startCraft()
if (result.isSuccess()) {
    updateQueueDisplay();  // 直接调用
}
```

### 2. 制作完成时

```java
// CraftScheduler.executeCraft()
plugin.getGuiListener().updatePlayerQueueDisplay(player);  // 直接调用
```

### 3. 定时刷新（剩余时间同步）

```java
// CustomForgeGUI.startUpdateTask()
updateTaskId = plugin.getServer().getScheduler().scheduleSyncRepeatingTask(
    plugin, () -> updateQueueDisplay(), 
    20L, 20L  // 每 20 tick (1秒) 执行一次
);
```

### 更新时机汇总

| 时机 | 更新方式 | 说明 |
|------|----------|------|
| 点击制作 | 直接调用 `updateQueueDisplay()` | 立即显示新加入的任务 |
| 制作完成 | 直接调用 `updatePlayerQueueDisplay()` | 立即显示任务完成和下一个任务 |
| 每 1 秒 | 定时任务 | 同步正在制作的物品剩余时间 |
