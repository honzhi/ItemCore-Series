## 📋 开发目标

根据用户布局配置的队列槽位顺序，映射显示队列中的物品。

---

## 🔧 设计方案

### 用户布局配置示例

```yaml
# layouts/Default.yml
layout:
  - 'AAAAAAAAA'  
  - 'AAA000AAA'  
  - 'AZA000AEA'  
  - 'AAAXXXAAA'  
  - 'AAAAAAAAA'  
  - '←AXXXXXAA'  

slots:
  X:
    material: GREEN_STAINED_GLASS_PANE
    function: queue_display
```

## 🛠 实现内容

### 1. 队列槽位顺序记录

**文件**: `gui/CustomForgeGUI.java`

```java
private final List<Integer> queueSlots = new ArrayList<>();

private void renderSlot(int slot, LayoutLoader.SlotConfig config) {
    String function = config.getFunction();
    slotFunctions.put(slot, function);
    
    // 按顺序记录队列槽位
    if ("queue_display".equals(function)) {
        queueSlots.add(slot);
    }
    
    // 根据队列槽位索引显示对应的队列物品
    switch (function) {
        case "queue_display" -> renderQueueDisplay(slot, config, queueSlots.size() - 1);
        // ... 其他功能
    }
}
```
