# 开发日志 - DEV-022 - 取消制作与领取功能

**开发日期**: 2026-05-27  
**开发阶段**: Phase 22  
**开发状态**: ✅ 已完成

## 📋 开发目标

1. 修复制作完成时立即刷新的问题（物品已到背包还残留显示）
2. 添加取消制作功能
3. 修改制作完成逻辑：玩家点击领取后才给物品

---

## 🔧 问题分析

### 问题 1: 制作完成后残留显示

**原逻辑**:
```java
// executeCraft() 中
giveOutput(player, recipe);  // 直接给玩家
queue.completeCurrentTask(); // 立即移除任务
```

**问题**:
- `completeCurrentTask()` 立即从队列中移除任务
- 虽然调用了 `updateQueueDisplay()`，但任务已经不在队列中了
- 所以刷新后该槽位显示为空
- 但如果有定时刷新（1秒后），会显示空槽位

### 问题 2: 无法取消制作

**原逻辑**:
- 没有取消制作的方法
- `CraftTask.cancel()` 只是标记 cancelled，但没有实际功能

### 问题 3: 制作完成后直接给物品

**原逻辑**:
- 制作完成后立即给玩家物品
- 玩家无法选择何时领取

---

## 🛠 解决方案

### 方案 1: 任务状态机

在 `CraftTask` 中引入 `TaskStatus` 枚举：

```java
public enum TaskStatus {
    CRAFTING,   // 制作中
    COMPLETED,  // 已完成（待领取）
    CLAIMED     // 已领取
}
```

### 方案 2: 队列重构

修改 `RecipeQueue`，将任务分为三部分：
- `currentTask`: 当前正在制作的任务
- `waitingTasks`: 等待中的任务
- `completedTasks`: 已完成待领取的任务

**关键方法**:
- `markTaskCompleted(taskId)`: 将任务标记为 COMPLETED，移到 completedTasks
- `removeClaimedTask(taskId)`: 从 completedTasks 中移除任务
- `cancelTask(taskId)`: 取消任务，从队列中移除

### 方案 3: 取消制作和领取功能

在 `CraftScheduler` 中添加：
- `cancelCraft(player, forgeId, taskId)`: 取消制作，返还材料
- `claimItem(player, forgeId, taskId)`: 领取已完成的物品

### 方案 4: 背包已满检查

在 `MaterialChecker` 中添加：
- `canGiveMaterials(player, materials)`: 检查背包是否有空间
- `giveMaterials(player, materials)`: 返还材料给玩家

```java
public class CraftTask {
    public enum TaskStatus {
        CRAFTING,
        COMPLETED,
        CLAIMED
    }

    private TaskStatus status = TaskStatus.CRAFTING;

    public TaskStatus getStatus() {
        return status;
    }

    public void setStatus(TaskStatus status) {
        this.status = status;
    }

    public boolean isCompleted() {
        return status == TaskStatus.COMPLETED;
    }

    public boolean isClaimed() {
        return status == TaskStatus.CLAIMED;
    }
}
```

### 2. 修改 RecipeQueue

**文件**: `core/RecipeQueue.java`

```java
public class RecipeQueue {
    private CraftTask currentTask;
    private final Queue<CraftTask> waitingTasks = new LinkedList<>();
    private final List<CraftTask> completedTasks = new ArrayList<>();

    public void markTaskCompleted(UUID taskId) {
        if (currentTask != null && currentTask.getTaskId().equals(taskId)) {
            currentTask.setStatus(CraftTask.TaskStatus.COMPLETED);
            completedTasks.add(currentTask);
            currentTask = null;
            startNextTask();
        }
        // ... 处理等待中的任务
    }

    public void removeClaimedTask(UUID taskId) {
        Iterator<CraftTask> it = completedTasks.iterator();
        while (it.hasNext()) {
            CraftTask task = it.next();
            if (task.getTaskId().equals(taskId)) {
                task.setStatus(CraftTask.TaskStatus.CLAIMED);
                it.remove();
                break;
            }
        }
    }

    public boolean cancelTask(UUID taskId) {
        if (currentTask != null && currentTask.getTaskId().equals(taskId)) {
            if (currentTask.getStatus() == CraftTask.TaskStatus.CRAFTING) {
                currentTask.cancel();
                currentTask = null;
                startNextTask();
                return true;
            }
            return false;
        }
        // ... 处理等待中的任务
    }

    public List<CraftTask> getAllTasks() {
        List<CraftTask> allTasks = new ArrayList<>();
        if (currentTask != null) {
            allTasks.add(currentTask);
        }
        allTasks.addAll(waitingTasks);
        allTasks.addAll(completedTasks);
        return allTasks;
    }
}
```

#### 新增 cancelCraft()

```java
public CraftResult cancelCraft(Player player, String forgeId, UUID taskId) {
    RecipeQueue queue = plugin.getPlayerQueueManager().getQueue(player, forgeId);
    CraftTask task = queue.getTask(taskId);
    
    if (task == null) {
        return CraftResult.failure("任务不存在");
    }

    if (task.isCompleted()) {
        return CraftResult.failure("任务已完成，无法取消");
    }

    // 检查背包空间
    MaterialChecker materialChecker = new MaterialChecker();
    if (!materialChecker.canGiveMaterials(player, recipe.getMaterials())) {
        return CraftResult.failure("背包已满，请清理背包后再取消制作");
    }

    // 取消任务
    if (task.getStatus() == CraftTask.TaskStatus.CRAFTING) {
        BukkitTask bukkitTask = activeTasks.remove(taskId);
        if (bukkitTask != null) {
            bukkitTask.cancel();
        }
        taskEndTimes.remove(taskId);
    }

    queue.cancelTask(taskId);

    // 返还材料
    materialChecker.giveMaterials(player, recipe.getMaterials());

    return CraftResult.success(task);
}
```

#### 新增 claimItem()

```java
public CraftResult claimItem(Player player, String forgeId, UUID taskId) {
    RecipeQueue queue = plugin.getPlayerQueueManager().getQueue(player, forgeId);
    CraftTask task = queue.getTask(taskId);
    
    if (task == null) {
        return CraftResult.failure("任务不存在");
    }

    if (!task.isCompleted()) {
        return CraftResult.failure("任务尚未完成");
    }

    // 检查背包空间
    MaterialChecker materialChecker = new MaterialChecker();
    if (!materialChecker.canGiveMaterials(player, recipe.getMaterials())) {
        return CraftResult.failure("背包已满，请清理背包后再领取");
    }

    // 给玩家物品
    giveOutput(player, recipe);
    queue.removeClaimedTask(taskId);

    return CraftResult.success(task);
}
```

```java
    if (task.isCompleted()) {
        claimItem(task);

        cancelTask(task);
```

### 任务状态流转

```
点击制作
    ↓
CRAFTING（制作中）
    ↓
制作完成
    ↓
COMPLETED（已完成，显示"点击领取"）
    ↓
点击领取
    ↓
CLAIMED（已领取，从队列移除）
```

### 取消制作流程

```
点击取消（等待中/制作中）
    ↓
检查背包空间
    ↓
有空间 → 返还材料 → 任务取消
    ↓
无空间 → 提示"背包已满，请清理背包后再取消"
```
