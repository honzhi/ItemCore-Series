## 🔧 核心实现

### 1. 制作流程

```
玩家点击确认
    ↓
触发 RecipePreCraftEvent
    ↓
检查条件 (ConditionChecker)
    ↓
检查材料 (MaterialChecker)
    ↓
消耗材料
    ↓
创建 CraftTask
    ↓
加入制作队列 (RecipeQueue)
    ↓
触发 RecipeQueueEvent
    ↓
异步制作中...
    ↓
制作完成
    ↓
发放物品
    ↓
触发 RecipeCraftedEvent
```

### 2. CraftResult 结果封装

```java
public class CraftResult {
    public enum ResultType {
        SUCCESS,           // 制作成功
        QUEUE_FULL,        // 队列已满
        CONDITION_FAILED,  // 条件不满足
        MATERIAL_INSUFFICIENT,  // 材料不足
        UNKNOWN_ERROR      // 未知错误
    }
    
    private ResultType type;
    private String message;
    private Forge.Recipe recipe;
}
```

### 3. 事件系统

**RecipePreCraftEvent**:
- 可取消的事件
- 包含玩家、配方、任务信息
- 允许其他插件阻止制作

**RecipeCraftedEvent**:
- 包含玩家、配方、完成时间
- 用于通知其他插件

**RecipeQueueEvent**:
- 队列添加/移除时触发
- 包含队列变化信息

## 📝 使用示例

```java
// 触发制作
Forge forge = plugin.getForgeLoader().getForge("铁匠铺");
Forge.Recipe recipe = forge.getRecipeById("iron_sword");

CraftResult result = plugin.getForgeManager().startCraft(player, forge, recipe, 1);
if (result.getType() == CraftResult.ResultType.SUCCESS) {
    player.sendMessage("制作已加入队列！");
}
```
