## 🔧 核心实现

### 1. ConditionChecker 条件检查器

```java
public class ConditionChecker {
    // 检查所有条件
    public CheckResult checkAll(Player player, List<Forge.Condition> conditions);
    
    // 检查等级条件
    private boolean checkLevel(Player player, Forge.Condition condition);
    
    // 检查权限条件
    private boolean checkPermission(Player player, Forge.Condition condition);
    
    // 检查经济条件
    private boolean checkMoney(Player player, Forge.Condition condition);
}
```

**条件类型**:
- `level`: 检查玩家等级
- `permission`: 检查玩家权限节点
- `money`: 检查玩家经济余额（需要 Vault）

### 2. MaterialChecker 材质验证器

```java
public class MaterialChecker {
    // 检查玩家物品栏中的材料
    public MaterialCheckResult checkMaterials(Player player, List<Forge.ItemReference> materials);
    
    // 消耗玩家物品栏中的材料
    public boolean consumeMaterials(Player player, List<Forge.ItemReference> materials);
    
    // 获取物品引用对应的实际物品
    private ItemStack resolveItem(Forge.ItemReference ref);
}
```

**检查结果**:
- `AVAILABLE`: 材料充足
- `INSUFFICIENT`: 材料不足
- `NOT_FOUND`: 材料不存在
