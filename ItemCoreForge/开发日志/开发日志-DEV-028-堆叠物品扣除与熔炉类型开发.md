### 1. CraftingTableRecipeManager.java - 堆叠物品扣除

**文件**: `recipe/CraftingTableRecipeManager.java`

#### 参考 GreatCrafts 插件的实现方式

GreatCrafts 通过以下方式实现堆叠物品的扣除：

1. **注册配方时**：只注册"基础配方"（每个槽位 1 个物品）
2. **事件拦截时**：
   - 调用 `event.setCancelled(true)` 取消原版行为
   - 自己计算最大可制作次数
   - 自己手动扣除材料（支持堆叠）
   - 自己手动给予输出物品

#### 核心实现

**配方注册时**：

```java
private void registerShapelessRecipe(...) {
    for (Forge.ItemReference material : recipe.getMaterials()) {
        RecipeChoice choice = createRecipeChoice(material);
        if (choice != null) {
            // 不拆分 amount，每个材料只添加 1 次（基础配方）
            shapelessRecipe.addIngredient(choice);
        }
    }
}
```

**事件处理时**：

```java
@EventHandler(priority = EventPriority.HIGHEST)
public void onCraftItem(CraftItemEvent event) {
    // 检查是否是我们注册的配方
    if (!key.getNamespace().equals(plugin.getName().toLowerCase())) {
        return;
    }

    // 取消原版行为
    event.setCancelled(true);

    // 计算最大可制作次数
    int maxCraftsAvailable = Integer.MAX_VALUE;
    for (Forge.ItemReference reqMat : requiredMaterials) {
        int highestCount = 0;
        for (int slotNum = 0; slotNum < 9; slotNum++) {
            ItemStack invItem = inventorySlotItems.get(slotNum);
            if (invItem != null && matchesMaterial(invItem, reqMat)) {
                if (highestCount < invItem.getAmount()) {
                    highestCount = invItem.getAmount();
                }
            }
        }
        int craftsPossible = highestCount / reqMat.getAmount();
        maxCraftsAvailable = Math.min(maxCraftsAvailable, craftsPossible);
    }

    // 根据点击类型决定实际制作数量
    int actualAmountCrafted;
    if (event.getClick() == ClickType.SHIFT_LEFT || event.getClick() == ClickType.SHIFT_RIGHT) {
        actualAmountCrafted = performShiftClickCraft(player, result, maxCraftsAvailable);
    } else {
        actualAmountCrafted = 1;
        player.setItemOnCursor(result.clone());
    }

    // 手动扣除材料
    for (int slotIndex = 0; slotIndex < 9; slotIndex++) {
        ItemStack stack = event.getInventory().getItem(slotIndex + 1);
        if (stack != null && stack.getType() != Material.AIR) {
            int requiredAmount = requiredMaterials.get(requiredSlotMapping[slotIndex]).getAmount();
            int removeAmount = requiredAmount * actualAmountCrafted;
            stack.setAmount(stack.getAmount() - removeAmount);
        }
    }
}
```

#### 贪婪匹配算法

无序配方使用贪婪匹配算法：

```java
// 按 amount 降序排序，优先匹配需要数量多的材料
requiredMaterials.sort((a, b) -> Integer.compare(b.getAmount(), a.getAmount()));

for (int reqIndex = 0; reqIndex < requiredMaterials.size(); reqIndex++) {
    Forge.ItemReference reqMat = requiredMaterials.get(reqIndex);
    int highestCount = 0;
    int highestCountSlot = -1;
    for (int slotNum = 0; slotNum < 9; slotNum++) {
        ItemStack invItem = inventorySlotItems.get(slotNum);
        if (invItem != null && matchesMaterial(invItem, reqMat)) {
            if (highestCount < invItem.getAmount()) {
                highestCount = invItem.getAmount();
                highestCountSlot = slotNum;
            }
        }
    }
    // 记录映射关系
    inventorySlotItems.set(highestCountSlot, null);
    requiredSlotMapping[highestCountSlot] = reqIndex;
}
```

### 4. FurnaceRecipeManager.java - 熔炉配方管理器（新增）

**文件**: `recipe/FurnaceRecipeManager.java`

#### 功能列表

| 功能 | 说明 |
|------|------|
| 注册配方 | 将配置中的配方注册到原版熔炉 |
| 条件检查 | 制作前检查玩家是否满足条件 |
| 事件监听 | 监听 `FurnaceSmeltEvent`，处理自定义配方 |

#### 核心实现

**注册配方**：

```java
private void registerRecipe(Forge forge, Forge.Recipe recipe) {
    if (recipe.getInput() == null || recipe.getOutput() == null) {
        return;
    }

    NamespacedKey key = new NamespacedKey(plugin, forge.getForgeId() + "/" + recipe.getRecipeId());
    ItemStack result = materialChecker.resolveItem(recipe.getOutput());
    RecipeChoice choice = createRecipeChoice(recipe.getInput());

    // 注册熔炉配方
    FurnaceRecipe furnaceRecipe = new FurnaceRecipe(key, result, choice, 0, 200);
    Bukkit.addRecipe(furnaceRecipe);
}
```

**事件处理**：

```java
@EventHandler(priority = EventPriority.HIGHEST)
public void onFurnaceSmelt(FurnaceSmeltEvent event) {
    // 检查是否是我们注册的配方
    if (!key.getNamespace().equals(plugin.getName().toLowerCase())) {
        return;
    }

    // 检查附近玩家的条件
    Block furnaceBlock = event.getBlock();
    for (Entity entity : furnaceBlock.getWorld().getNearbyEntities(
            furnaceBlock.getLocation(), 5, 5, 5)) {
        if (entity instanceof Player) {
            Player player = (Player) entity;
            if (!conditionChecker.allConditionsMet(player, forgeRecipe.getConditions())) {
                event.setCancelled(true);
                MessageUtil.sendMessage(player, "conditions-not-met");
                return;
            }
        }
    }
}
```

### 5. FurnaceHandler.java - 熔炉类型处理器（新增）

**文件**: `type/furnace/FurnaceHandler.java`

```java
public class FurnaceHandler implements ForgeTypeHandler {
    @Override
    public String getType() {
        return "furnace";
    }

    @Override
    public void onEnable(ItemCoreForge plugin) {
        this.recipeManager = new FurnaceRecipeManager(plugin);
        this.recipeManager.setMaterialChecker(plugin.getMaterialChecker());
        this.recipeManager.registerEvents();
    }

    @Override
    public void onDisable() {
        if (recipeManager != null) {
            recipeManager.unregisterAll();
        }
    }

    @Override
    public void registerForge(Forge forge) {
        if (recipeManager != null) {
            recipeManager.registerForgeRecipes(forge);
        }
    }
}
```

## 🚀 使用示例

### 1. 配置堆叠物品扣除

```yaml
# crafting_table 类型
recipes:
  iron_sword:
    output:
      source: "vanilla"
      id: "IRON_SWORD"
      amount: 1
    materials:
      - source: "vanilla"
        id: "IRON_INGOT"
        amount: 2    # 需要 2 个铁锭（支持堆叠扣除）
      - source: "vanilla"
        id: "STICK"
        amount: 1
```

**玩家行为**：

| 操作 | 玩家有 64 个铁锭 | 扣除数量 | 制作数量 |
|------|-----------------|---------|---------|
| 普通点击 | 64 个 | 2 个 | 1 个 |
| Shift 点击 | 64 个 | 64 个 | 32 个（64/2） |
