## 🛠 修复内容

### 1. 修改 QueueDisplayGUI

**文件**: `gui/QueueDisplayGUI.java`

```java
// 添加队列槽位列表字段
private final List<Integer> queueSlots;

// 构造函数接收队列槽位参数
public QueueDisplayGUI(ItemCoreForge plugin, Player player, Forge forge, List<Integer> queueSlots) {
    super(plugin, player, "制作队列 - " + forge.getDisplayName(), 6);
    this.forge = forge;
    this.queueSlots = queueSlots != null ? new ArrayList<>(queueSlots) : getQueueSlotsFromLayout(plugin, forge);
    render();
    startUpdateTask();
}

// 从布局中获取队列槽位
private List<Integer> getQueueSlotsFromLayout(ItemCoreForge plugin, Forge forge) {
    List<Integer> slots = new ArrayList<>();
    String layoutName = forge.getLayoutFile().replace(".yml", "");
    LayoutLoader.Layout layout = plugin.getLayoutLoader().getLayout(layoutName);
    
    if (layout != null) {
        for (int row = 0; row < layout.getRows(); row++) {
            for (int col = 0; col < 9; col++) {
                int slot = row * 9 + col;
                char c = layout.getSlot(row, col);
                LayoutLoader.SlotConfig config = layout.getSlotConfig(c);
                
                if (config != null && "queue_display".equals(config.getFunction())) {
                    slots.add(slot);
                }
            }
        }
    }
    
    return slots;
}
```
