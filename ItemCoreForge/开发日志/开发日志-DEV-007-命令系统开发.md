## 🔧 核心实现

### 1. 命令结构

| 命令 | 权限 | 参数 | 说明 |
|------|------|------|------|
| `/icf` | `itemcoreforge.use` | `<forge-id>` | 打开指定锻造台 |
| `/icf list` | `itemcoreforge.list` | 无 | 列出所有锻造台 |
| `/icf reload` | `itemcoreforge.reload` | 无 | 重载配置文件 |

### 2. 命令执行器

```java
public class ForgeCommand implements CommandExecutor {
    
    @Override
    public boolean onCommand(CommandSender sender, Command command, 
                           String label, String[] args) {
        // 无参数：显示帮助
        if (args.length == 0) {
            return showHelp(sender);
        }
        
        // 子命令处理
        switch (args[0].toLowerCase()) {
            case "list":
                return listForges(sender);
            case "reload":
                return reloadConfig(sender);
            default:
                // 尝试打开锻造台
                return openForge(sender, args[0]);
        }
    }
}
```

### 3. 权限定义

```yaml
permissions:
  itemcoreforge.use:
    description: 打开锻造台
    default: true
  itemcoreforge.list:
    description: 列出所有锻造台
    default: op
  itemcoreforge.reload:
    description: 重载配置文件
    default: op
```
