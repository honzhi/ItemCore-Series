修复制作队列不显示的问题，实现：

2. 实时更新剩余时间
3. 显示正在制作和等待制作的物品

---

## 🔧 问题分析

### 问题描述
- 用户打开锻造台 GUI，点击配方后队列中不显示任何物品
- 布局中有多个 `X` 槽位，但代码只处理单个队列显示

### 根本原因
1. `renderQueueDisplay` 方法没有接收索引参数，无法区分多个队列槽位
2. 没有记录队列槽位的列表
3. 没有定时更新机制来刷新剩余时间

```java
// 添加索引参数
private void renderQueueDisplay(int slot, LayoutLoader.SlotConfig config, int queueIndex) {
    RecipeQueue queue = plugin.getPlayerQueueManager().getQueue(player, forge.getForgeId());
    List<CraftTask> allTasks = queue.getAllTasks();
    
    CraftTask task = null;
    if (queueIndex < allTasks.size()) {
        task = allTasks.get(queueIndex);
    }
    
    if (task == null) {
        // 显示空队列
    } else {
        // 显示任务信息
        if (queueIndex == 0) {
            meta.displayName(MessageUtil.toComponent("&a正在制作"));
        } else {
            meta.displayName(MessageUtil.toComponent("&e等待中 " + (queueIndex + 1)));
        }
    }
}
```

### 3. 添加定时更新机制

**修改文件**: `gui/CustomForgeGUI.java`

```java
// 新增字段
private int updateTaskId = -1;

// 在构造函数中启动定时任务
public CustomForgeGUI(ItemCoreForge plugin, Player player, Forge forge) {
    // ... 其他代码
    startUpdateTask();
}

// 启动更新任务
private void startUpdateTask() {
    updateTaskId = plugin.getServer().getScheduler().scheduleSyncRepeatingTask(plugin, () -> {
        if (player.isOnline() && inventory.getViewers().contains(player)) {
            updateQueueDisplay();
        } else {
            stopUpdateTask();
        }
    }, 20L, 20L); // 每秒更新一次
}

// 停止更新任务
private void stopUpdateTask() {
    if (updateTaskId != -1) {
        plugin.getServer().getScheduler().cancelTask(updateTaskId);
        updateTaskId = -1;
    }
}

// 重写 close 方法以停止任务
@Override
public void close() {
    stopUpdateTask();
    super.close();
}
```

### 4. 更新队列显示

**修改文件**: `gui/CustomForgeGUI.java`

```java
private void updateQueueDisplay() {
    for (int i = 0; i < queueSlots.size(); i++) {
        int slot = queueSlots.get(i);
        LayoutLoader.SlotConfig config = getSlotConfigForSlot(slot);
        if (config != null) {
            renderQueueDisplay(slot, config, i);
        }
    }
}

private LayoutLoader.SlotConfig getSlotConfigForSlot(int slot) {
    if (layout == null) return null;
    int row = slot / 9;
    int col = slot % 9;
    char c = layout.getSlot(row, col);
    return layout.getSlotConfig(c);
}
```
