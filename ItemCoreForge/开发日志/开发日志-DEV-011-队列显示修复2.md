## 🔧 问题分析

### 问题描述
- 队列槽位不显示正在制作的物品
- 定时更新任务无法正常刷新队列显示

### 根本原因
1. `Layout` 类缺少 `getSlotConfigs()` 方法，导致无法获取所有槽位配置
2. `updateQueueDisplay()` 方法中的逻辑有缺陷

---

```java
    return slotConfigs;

```

### 2. 修复队列更新逻辑

**修改文件**: `gui/CustomForgeGUI.java`

```java
private void updateQueueDisplay() {
    if (layout == null) return;
    
    // 找到队列显示的配置
    LayoutLoader.SlotConfig queueConfig = null;
    for (LayoutLoader.SlotConfig config : layout.getSlotConfigs().values()) {
        if ("queue_display".equals(config.getFunction())) {
            queueConfig = config;
            break;
        }
    }
    
    if (queueConfig == null) return;
    
    // 更新所有队列槽位
    for (int i = 0; i < queueSlots.size(); i++) {
        int slot = queueSlots.get(i);
        renderQueueDisplay(slot, queueConfig, i);
    }
}
```
