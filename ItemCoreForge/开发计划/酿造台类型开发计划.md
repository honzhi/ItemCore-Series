## 📋 概述

### 目标
实现 `brewing_stand` 类型锻造台，将配置的配方集成到原版酿造台中。玩家可以在原版酿造台中使用自定义配方酿造药水/物品。

### 与现有类型对比

| 类型 | 配置特点 | 主要事件 |
|------|---------|----------|
| `crafting_table` | `materials` 列表 | `CraftItemEvent` |
| `furnace` | `input` + `output` | `FurnaceSmeltEvent` |
| `brewing_stand` | `input`（药水瓶）+ `ingredient`（材料）+ `output`（结果） | `BrewingStartEvent` |

```yaml
    # 酿造材料（顶部槽位）
    ingredient:
      source: "vanilla"
      id: "GOLD_INGOT"
      
    # 输出物品（酿造完成后）
    output:
      source: "vanilla"
      id: "POTION"
      # 如果是自定义药水，需要配置 itemcore 物品
      
    # 酿造时间（tick，1 tick = 0.05 秒）

    brew-time: 400
    
    # 制作条件（可选）
    conditions:
      - type: "level"
        value: 5
        
  # 配方2：ItemCore 药水配方示例
  custom_potion_recipe:
    # 使用 ItemCore 自定义药水作为输入
    input:
      source: "itemcore"
      category: "potions"
      id: "magic_base_potion"

      source: "itemcore"
      category: "materials"
      id: "magic_dust"

    output:
      source: "itemcore"
```

| 字段 | 类型 | 必填 | 说明 |

### Step 3: 创建酿造台类型处理器

**目标**: 创建 `BrewingStandHandler` 实现 `ForgeTypeHandler`

```java
package com.minemart.itemcoreforge.type.brewingstand;

import com.minemart.itemcoreforge.ItemCoreForge;
import com.minemart.itemcoreforge.core.Forge;
import com.minemart.itemcoreforge.recipe.BrewingStandRecipeManager;
import com.minemart.itemcoreforge.type.ForgeTypeHandler;
import com.minemart.itemcoreforge.utils.DebugLogger;

public class BrewingStandHandler implements ForgeTypeHandler {

    private ItemCoreForge plugin;
    private BrewingStandRecipeManager recipeManager;

    @Override
    public String getType() {
        return "brewing_stand";
    }

    @Override
    public void onEnable(ItemCoreForge plugin) {
        this.plugin = plugin;
        this.recipeManager = new BrewingStandRecipeManager(plugin);
        this.recipeManager.setMaterialChecker(plugin.getMaterialChecker());
        this.recipeManager.registerEvents();
        DebugLogger.info("BrewingStand", "BrewingStand 类型处理器已启用");
    }

    @Override
    public void onDisable() {
        if (recipeManager != null) {
            recipeManager.unregisterAll();
            DebugLogger.info("BrewingStand", "BrewingStand 类型处理器已禁用");
        }
    }

    @Override
    public void onReload() {
        if (recipeManager != null) {
            recipeManager.unregisterAll();
            DebugLogger.info("BrewingStand", "BrewingStand 类型处理器已重载（配方已清除）");
        }
    }

    @Override
    public void registerForge(Forge forge) {
        if (recipeManager != null) {
            recipeManager.registerForgeRecipes(forge);
            DebugLogger.info("BrewingStand", "已注册锻造台到 BrewingStand: " + forge.getForgeId());
        }
    }
}
```

#### 4.2 注册配方方法

```java
private void registerRecipe(Forge forge, Forge.Recipe recipe) {
    // 校验必填字段
    if (recipe.getInput() == null || recipe.getIngredient() == null || recipe.getOutput() == null) {
        plugin.getLogger().warning("酿造台配方缺少必填字段: " + recipe.getRecipeId());
        return;
    }

    NamespacedKey key = new NamespacedKey(plugin, forge.getForgeId() + "/" + recipe.getRecipeId());

    // 解析输出物品
    ItemStack result = materialChecker.resolveItem(recipe.getOutput());
    if (result == null) {
        plugin.getLogger().warning("无法解析输出物品: " + recipe.getRecipeId());
        return;
    }

    DebugLogger.info("BrewingStand", "注册配方: " + key);

    // 创建 input 的 RecipeChoice（药水瓶）
    RecipeChoice inputChoice = createRecipeChoice(recipe.getInput());
    if (inputChoice == null) {
        plugin.getLogger().warning("无法解析输入物品: " + recipe.getInput().getId());
        return;
    }

    // 创建 ingredient 的 RecipeChoice（酿造材料）
    RecipeChoice ingredientChoice = createRecipeChoice(recipe.getIngredient());
    if (ingredientChoice == null) {
        plugin.getLogger().warning("无法解析酿造材料: " + recipe.getIngredient().getId());
        return;
    }

    // 创建并注册 PotionMix
    PotionMix potionMix = new PotionMix(key, result, inputChoice, ingredientChoice);
    Bukkit.getPotionBrewer().addPotionMix(potionMix);

    // 记录酿造时间（用于 BrewingStartEvent）
    recipeBrewTimes.put(key, recipe.getBrewTime());

    DebugLogger.info("BrewingStand", "  - 输入: " + recipe.getInput().getId() + 
                    " + 材料: " + recipe.getIngredient().getId() + 
                    " -> 输出: " + recipe.getOutput().getId());
}
```

#### 4.3 创建 RecipeChoice 方法

```java
private RecipeChoice createRecipeChoice(Forge.ItemReference material) {
    if (material.isItemCore()) {
        // ItemCore 物品使用 ExactChoice（精确匹配物品）
        ItemStack item = materialChecker.resolveItem(material);
        if (item != null) {
            return new RecipeChoice.ExactChoice(item);
        }
        return null;
    } else {
        // 原版物品使用 MaterialChoice（按材料类型匹配）
        try {
            Material mat = Material.matchMaterial(material.getId().toUpperCase());
            if (mat != null) {
                return new RecipeChoice.MaterialChoice(mat);
            }
        } catch (Exception e) {
            plugin.getLogger().warning("无效的材料: " + material.getId());
        }
        return null;
    }
}
```

#### 4.4 注销配方方法

```java
public void unregisterAll() {
    PotionBrewer brewer = Bukkit.getPotionBrewer();
    for (Forge forge : registeredRecipes.values()) {
        for (Forge.Recipe recipe : forge.getRecipes().values()) {
            NamespacedKey key = new NamespacedKey(plugin, forge.getForgeId() + "/" + recipe.getRecipeId());
            brewer.removePotionMix(key);
        }
    }
    registeredRecipes.clear();
    recipeBrewTimes.clear();
}
```

#### 4.5 事件监听（BrewingStartEvent）

```java
@EventHandler(priority = EventPriority.HIGHEST)
public void onBrewingStart(BrewingStartEvent event) {
    ItemStack source = event.getSource();  // 顶部材料
    BrewerInventory inventory = (BrewerInventory) event.getInventory();
    
    // 查找匹配的配方（遍历所有注册的配方）
    // 注意：PotionMix 内部匹配，我们需要通过另一种方式获取当前配方
    
    // 方案：遍历注册的配方，检查 input 和 ingredient 是否匹配
    for (Forge forge : registeredRecipes.values()) {
        if (!forge.isBrewingStandType()) {
            continue;
        }
        
        for (Forge.Recipe recipe : forge.getRecipes().values()) {
            if (matchesRecipe(inventory, recipe, source)) {
                // 找到匹配的配方
                NamespacedKey key = new NamespacedKey(plugin, forge.getForgeId() + "/" + recipe.getRecipeId());
                
                // 设置自定义酿造时间
                Integer brewTime = recipeBrewTimes.get(key);
                if (brewTime != null) {
                    event.setRecipeBrewTime(brewTime);
                    DebugLogger.info("BrewingStand", "设置酿造时间: " + brewTime + " ticks");
                }
                
                // 检查条件（检查附近玩家）
                org.bukkit.block.Block brewingStand = event.getBlock();
                for (org.bukkit.entity.Entity entity : brewingStand.getWorld().getNearbyEntities(
                        brewingStand.getLocation(), 5, 5, 5)) {
                    if (entity instanceof Player) {
                        Player player = (Player) entity;
                        if (!conditionChecker.allConditionsMet(player, recipe.getConditions())) {
                            event.setCancelled(true);
                            MessageUtil.sendMessage(player, "conditions-not-met");
                            return;
                        }
                    }
                }
                
                return;  // 找到匹配配方后返回
            }
        }
    }
}

// 辅助方法：检查配方是否匹配
private boolean matchesRecipe(BrewerInventory inventory, Forge.Recipe recipe, ItemStack source) {
    // 检查 ingredient（顶部材料）
    RecipeChoice ingredientChoice = createRecipeChoice(recipe.getIngredient());
    if (ingredientChoice == null || !ingredientChoice.test(source)) {
        return false;
    }
    
    // 检查 input（底部药水瓶）
    RecipeChoice inputChoice = createRecipeChoice(recipe.getInput());
    if (inputChoice == null) {
        return false;
    }
    
    // 至少有一个药水瓶匹配
    boolean hasMatch = false;
    for (int i = 0; i < 3; i++) {
        ItemStack bottle = inventory.getItem(i + 1);  // 槽位 1, 2, 3
        if (bottle != null && bottle.getType() != Material.AIR) {
            if (inputChoice.test(bottle)) {
                hasMatch = true;
                break;
            }
        }
    }
    
    return hasMatch;
}
```

## 📝 关键难点与解决方案

### 难点1: PotionMix 配方的事件关联

**问题**: `BrewingStartEvent` 没有直接提供配方的 NamespacedKey，无法像 `FurnaceSmeltEvent` 那样通过 `getRecipe()` 获取。

**解决方案**:
- 在事件中遍历所有注册的配方
- 通过 `RecipeChoice.test()` 方法检查 input 和 ingredient 是否匹配
- 找到匹配的配方后应用自定义设置

**备选方案**: 使用 `event.getSource()` 和 `event.getInventory()` 的内容进行匹配。

---

### 难点2: 自定义药水的处理

**问题**: ItemCore 中的自定义药水可能是普通物品（非 Potion 类型），无法放入酿造台。

**解决方案**:
- 使用 `RecipeChoice.ExactChoice` 精确匹配 ItemStack
- 对于非标准药水物品，可能需要在配置文件中明确说明限制
- 文档中注明：`input` 建议使用原版 POTION 类型物品或 ItemCore 中可放入酿造台的物品

---

### 难点3: 酿造时间设置

**问题**: `PotionMix` 构造函数不接受酿造时间参数。

**解决方案**:
- 通过 `BrewingStartEvent.setRecipeBrewTime(int)` 动态设置
- 在 `recipeBrewTimes` Map 中缓存每个配方的酿造时间
