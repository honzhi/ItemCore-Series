### 1. 代码修复

### 2. 插件构建

成功使用Maven构建插件：

```bash
mvn clean package
```

**生成的文件**：
- `target/itemcoretrinkets-1.0.0.jar` - 可部署的插件JAR文件

### 3. 依赖管理

将ItemCore依赖从远程JitPack仓库改为本地Maven仓库：

1. 先安装ItemCore到本地仓库：
   ```bash
   cd ItemCore
   mvn install -DskipTests
   ```

2. 修改pom.xml依赖配置：
   ```xml
   <dependency>
       <groupId>com.minemart</groupId>
       <artifactId>itemcore</artifactId>
       <version>1.0.0</version>
       <scope>provided</scope>
   </dependency>
   ```
