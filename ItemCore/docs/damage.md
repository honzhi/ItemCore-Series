# 伤害系统

## 伤害类型

| 类型 | 标签 | 说明 |
|------|------|------|
| 物理 | `PHYSICAL` | 普攻默认 |
| 法术 | `SPELL` | 技能默认 |
| 射弹 | `PROJECTILE` | 射弹伤害 |

## 计算流程

```
原始伤害
  → 攻击类型标签
  → 暴击判定
  → 伤害类型百分比加成
  → 穿透计算
  → 抗性减免（非元素伤害）
  → 元素抗性减免（元素伤害）
  → 伤害减免 DAMAGE_REDUCTION%
  → 元素积累
  → 最终伤害
```

## 元素伤害规则

- 带元素的伤害跳过物理/法术抗性
- 只受元素抗性和 DAMAGE_REDUCTION 影响
- 负元素抗性 = 易伤加成

## 伤害减免

`DAMAGE_REDUCTION` 对所有伤害生效（包括元素伤害），为最终减伤。
