# 常见问题

**Q: 修改配置后需要重启服务器吗？**
A: `/ic reload` 即可热重载。

**Q: 属性为什么不生效？**
A: 检查 `active-slots` 配置。

**Q: 多装备属性如何计算？**
A: 同名属性自动累加。

**Q: 暴击伤害显示 200%？**
A: 总暴击 = 默认值（150%）+ `CRIT_DAMAGE`。

**Q: 攻击速度怎么配置？**
A: 直接写最终值，`1.6` = 铁剑速度。

**Q: 百分比属性怎么写？**
A: 直接填整数，`30` = 30%。

**Q: 适应之力有什么用？**
A: 自动转为攻击或法强（取高者）。

**Q: 元素伤害怎么计算？**
A: 跳过物抗/法抗，只受元素抗性和伤害减免影响。

**Q: 技能怎么触发？**
A: 安装 ItemCoreMythic，配置 `skills` 节点。

**Q: 创造模式物品攻速不对？**
A: 创造模式物品栏机制问题，生存模式正常。
