## 问题分析

### 原始问题
盔甲架在显示伤害数字时会出现短暂的可见闪烁，原因是：
1. 先调用 `spawn()` 生成实体
2. 然后才设置 `setVisible(false)` 等属性
3. 这两个操作之间有时间差，导致闪烁

### 解决方案
使用 Bukkit 提供的 `World.spawn(Location, Class, Consumer)` 方法，在实体生成前通过 Consumer 设置所有属性。

```java
// 修复后的代码
ArmorStand armorStand = target.getWorld().spawn(
    target.getLocation().add(0, height, 0),
    ArmorStand.class,
    as -> {
        // 在生成前设置所有属性
        as.setGravity(false);
        as.setInvulnerable(true);
        as.setVisible(false);
        as.setMarker(true);
        as.setSmall(true);
        as.setCustomNameVisible(true);
        as.setCustomName(damageText);
    }
);
```

## 额外优化

### 1. Marker 属性
```java
as.setMarker(true);
```
- 使盔甲架完全不可碰撞
- 减少实体的性能开销

### 2. Small 属性
```java
as.setSmall(true);
```
- 使盔甲架尺寸更小
- 更适合显示伤害数字
